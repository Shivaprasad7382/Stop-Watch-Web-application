# ⏱ Advanced Stopwatch Web Application

## Features
- Start / Pause / Reset stopwatch
- Lap tracking
- LocalStorage persistence
- Dark / Light mode
- Keyboard shortcuts
- Responsive design

## Technologies
- HTML
- CSS
- JavaScript

## Keyboard Shortcuts
- Space → Start / Pause
- L → Add Lap
- R → Reset

## Future Enhancements
- Export laps as CSV
- Countdown timer
- PWA support
